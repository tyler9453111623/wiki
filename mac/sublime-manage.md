## 快捷键

Ctrl + P，上一行
Ctrl + N，下一行
Ctrl + D，删除这行

cmd+shift+p,  包安装 > 输入：install package > 输入：packgeName  
cmd+alt+f，   查找替换  
⌘⌃上下键      两行交换位置  


## 包管理
首选项 -> package control  
Cmd + Shift + p 启用控制台 -> 输入：install package  

## 主题颜色管理
Cmd + Shift + p 启用控制台 -> 输入：install package -> 输入：Colorsublime(所有主题) -> Esc退出  
Cmd + Shift + p 启用控制台 -> 输入：select color schema -> 方向键选择主题
